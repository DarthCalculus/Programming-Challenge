You have 6 symbols:

+ Increment highlighted Value

- Decrement highlighted Value

> Move Pointer Right

< Move Pointer Left

[ Begin Loop

] Go to the correspoding begin statement if the highlighted value is not 0

In addition, you can use . for debugging to print the current cell and value.

Other characters are ignored

Challenges:

1. Obtain 15 using 14 characters

2. Put 10 in each cell using 40 characters

3. How big of a number can you get using 30 characters?

BONUS CHALLENGE: Obtain 16 using 13 characters

Tips:

Try moving left when at the leftmost box.

No negative numbers supported!
