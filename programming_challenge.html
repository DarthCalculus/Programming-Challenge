PROGRAMMING CHALLENGE

+ Increment highlighted Value

- Decrement highlighted Value

> Move Pointer Right

< Move Pointer Left

[ Begin Loop

] Go to the correspoding begin statement if the highlighted value is not 0

Other characters are ignored

2. Put 10 in each cell using 40 characters

3. How big of a number can you get using 30 characters?
